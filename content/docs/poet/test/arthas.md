# Arthas

## 功能大纲

- **thread**：用于排查线程死锁、CPU过高等问题。
- **sc**：用于查找JVM加载的类信息，`sm`可以查找JVM加载的函数信息，可以用于排查线上jar包版本是否正确。
- **jad**：用于反编译代码。
- **watch**：用于监测指定方法的参数、返回值和异常信息。
- **trace**：用于监测方法的调用链，展示每个调用的耗时，找出最耗时的方法。
- **profiler**：用于生成火焰图，监测CPU运行情况和函数调用频率。
- **ognl**：用于执行动态代码。
